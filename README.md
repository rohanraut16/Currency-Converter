# 💱 Currency Converter

A simple and responsive Currency Converter built using **HTML, CSS, and JavaScript**. It fetches real-time exchange rates from the ExchangeRate API and converts currencies instantly.

## 🔗 Live Demo

https://rohanraut16.github.io/Currency-Converter/

## 🚀 Features

* Convert currencies in real-time
* Supports multiple currencies:

  * USD (US Dollar)
  * INR (Indian Rupee)
  * EUR (Euro)
  * GBP (British Pound)
  * JPY (Japanese Yen)
* Live exchange rate display
* Clean and responsive UI
* Fast conversion using ExchangeRate API

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* ExchangeRate API

## ⚙️ How It Works

1. Enter an amount.
2. Select the source currency.
3. Select the target currency.
4. Click the **Convert** button.
5. The application fetches the latest exchange rate and displays:

   * Converted amount
   * Current exchange rate
